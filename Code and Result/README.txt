Pre-Requisites:

To view/run the code in Jupyter Notebook:
  1. Open Anaconda Navigator. Launch Jupyter.
  2. Upload the MLProject_SMSClassification.ipynb and spam.csv files.
  3. Open MLProject_SMSClassification.ipynb. From 'Cell' option in the menu bar, click “Run All”.
